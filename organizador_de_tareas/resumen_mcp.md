## Soporte Multilenguaje
La clase principal `Organizator` está diseñada con un sistema de internacionalización (i18n) integrado. Actualmente, el proyecto soporta tres idiomas, con diccionarios de traducción completos para toda la interfaz de usuario:

- **Español (es):** Idioma por defecto.
- **Portugués (pt):** Soporte completo para el público de Brasil/Portugal.
- **Inglés (en):** Para alcance global.

La función `setLanguage()` permite al usuario cambiar el idioma de la aplicación en tiempo real, actualizando dinámicamente botones, placeholders, alertas y filtros sin necesidad de recargar la página. Todos los textos se gestionan a través del objeto `I18N`.
